java8-example
=============

This is the source code for the blog post 

http://exampledriven.wordpress.com/2014/10/03/java-8-date-time-api-example/


This example covers the following :

- Lambda expressions
    - Method references
    - Functional interfaces
- Stream
    - Infinite stream
    - Parallel stream
    - Terminal operation
    - Filter
- Time
    - Calendar.Builder
    - LocalDate
- String
    - String.join
    - StringJoiner
